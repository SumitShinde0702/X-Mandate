Overview

XAG is a TypeScript SDK for autonomous agents to interact with the XRP Ledger (XRPL). It provides identity management, escrow-based payments, and reputation tracking.

Installation

npm install xrpl dotenv

Quick Start

import { XAG } from './src/XAG'; const xag = new XAG(); // Connects to XRPL Testnet by default await xag.connect();

Constructor

constructor(network?: string)

Creates a new XAG instance.

  • network string (optional): XRPL network URL. Default: 'wss://s.altnet.rippletest.net:51233' (Testnet)
const xag = new XAG(); // or for mainnet: const xag = new XAG('wss://xrplcluster.com');

createAgent()

Promise<Agent> createAgent(config: AgentConfig)

Creates a new agent with XLS-40 DID registration.

Parameters

  • config.name string: Agent name
  • config.type 'buyer' | 'seller' | 'supplier' | 'consumer': Agent type
  • config.didMethod string (optional): DID method, defaults to 'xls-40'
  • config.seed string (optional): Wallet seed, auto-generated if not provided

Returns

  • address: XRPL address
  • seed: Wallet seed (keep secure!)
  • did: Decentralized Identifier (DID)
  • config: Original config

Example

const agent = await xag.createAgent({ name: "SolarMonitor-01", type: "supplier", didMethod: "xls-40" }); console.log(agent.did); // did:xrpl:1:r...

initiateTrade()

Promise<TradeResult> initiateTrade(config: TradeConfig, buyerSeed?: string)

Initiates a trade with escrow-locked payments (XRP) or direct payment (RLUSD).

Parameters

  • config.buyer string: Buyer DID or address
  • config.seller string: Seller DID or address
  • config.amount number: Amount to trade
  • config.token 'RLUSD' | 'XRP': Token type
  • config.condition string (optional): Cryptographic condition
  • config.finishAfter number (optional): Ripple epoch timestamp for time-lock
  • config.cancelAfter number (optional): Ripple epoch timestamp for cancellation
  • config.memo string (optional): Transaction memo for auditability
  • buyerSeed string (optional): Buyer wallet seed

Returns

  • hash: Transaction hash
  • sequence: Transaction sequence
  • amount: Trade amount
  • token: Token type
  • buyer: Buyer address
  • seller: Seller address

Example

const settlement = await xag.initiateTrade({ buyer: buyerDID, seller: sellerDID, amount: 100, token: "RLUSD", condition: fulfillmentCryptoCondition, memo: "Solar panel purchase" }, buyerAgent.seed);

Note: For XRP, uses native XRPL Escrow (time-locked). For RLUSD, uses Payment transaction (immediate).

fulfillTrade()

Promise<string> fulfillTrade(escrowHash: string, sellerDID: string, token?: 'RLUSD' | 'XRP', sellerSeed?: string)

Fulfills a trade by completing an escrow (XRP) or verifying payment (RLUSD).

Parameters

  • escrowHash string: Transaction hash of the escrow/payment
  • sellerDID string: Seller DID or address
  • token 'RLUSD' | 'XRP' (optional): Token type, defaults to 'XRP'
  • sellerSeed string (optional): Seller wallet seed

Returns

Transaction hash of the fulfillment

Example

const fulfillmentHash = await xag.fulfillTrade( trade.hash, sellerAgent.did, "XRP", sellerAgent.seed );

getReputation()

Promise<ReputationResult> getReputation(agentDID: string)

Gets reputation score for an agent based on on-chain transaction history.

Parameters

  • agentDID string: Agent DID or address

Returns

  • did: Agent DID
  • address: XRPL address
  • score: Reputation score (10 points per successful trade)
  • successfulTrades: Total successful trades
  • escrowCreates: Number of successful EscrowCreate transactions
  • escrowFinishes: Number of successful EscrowFinish transactions
  • payments: Number of successful Payment transactions

Example

const score = await xag.getReputation(agentDID); console.log(`Reputation: ${score.score} points`); console.log(`Successful trades: ${score.successfulTrades}`);

log()

Promise<string> log(message: string, level?: 'info' | 'success' | 'warn' | 'error', agentDID: string, agentSeed?: string)

Logs a message to the blockchain using XRPL Transaction Memos. Creates an immutable audit trail for A2A communication.

Parameters

  • message string: Log message
  • level 'info' | 'success' | 'warn' | 'error' (optional): Log level, defaults to 'info'
  • agentDID string: Agent DID or address
  • agentSeed string (optional): Agent wallet seed

Returns

Transaction hash of the log entry

Example

await xag.log("Agent interaction started", "info", agent.did, agent.seed); await xag.log("Trade completed successfully", "success", agent.did, agent.seed); await xag.log("Warning: Low balance", "warn", agent.did, agent.seed); await xag.log("Error: Connection failed", "error", agent.did, agent.seed);

Note: Uses XRPL Transaction Memos to store log data. Creates a Payment transaction to a sink address (1 drop = 0.000001 XRP) to ensure the log is permanently recorded on-chain.

getLogs()

Promise<Array<LogEntry>> getLogs(agentDID: string, limit?: number)

Retrieves logs from blockchain for an agent by parsing transaction memos.

Parameters

  • agentDID string: Agent DID or address
  • limit number (optional): Maximum number of logs, defaults to 50

Returns

  • message: Log message
  • level: Log level
  • timestamp: ISO timestamp
  • txHash: Transaction hash

Example

const logs = await xag.getLogs(agent.did, 20); logs.forEach(log => { console.log(`[${log.level}] ${log.message} - ${log.timestamp}`); console.log(`Transaction: ${log.txHash}`); });

getTransactionHistory()

Promise<Array<Transaction>> getTransactionHistory(agentDID: string, limit?: number)

Gets transaction history for an agent.

Parameters

  • agentDID string: Agent DID or address
  • limit number (optional): Maximum number of transactions, defaults to 50

Returns

  • hash: Transaction hash
  • type: Transaction type (e.g., 'Payment', 'EscrowCreate')
  • result: Transaction result (e.g., 'tesSUCCESS')
  • timestamp: ISO timestamp (if available)

Example

const history = await xag.getTransactionHistory(agent.did, 100); history.forEach(tx => { console.log(`${tx.type}: ${tx.result} - ${tx.hash}`); if (tx.timestamp) { console.log(`Time: ${tx.timestamp}`); } });

connect() & disconnect()

Promise<void> connect()
Promise<void> disconnect()

Connects to or disconnects from the XRPL network.

await xag.connect(); // ... use XAG ... await xag.disconnect();

Complete Example

import { XAG } from './src/XAG'; async function example() { const xag = new XAG(); try { // 1. Create agents const buyer = await xag.createAgent({ name: "AutoBuyer", type: "consumer", didMethod: "xls-40" }); const seller = await xag.createAgent({ name: "SolarSupplier", type: "supplier", didMethod: "xls-40" }); // 2. Initiate trade const trade = await xag.initiateTrade({ buyer: buyer.did, seller: seller.did, amount: 100, token: "XRP", memo: "Solar panel purchase" }, buyer.seed); // 3. Wait for time-lock (if XRP escrow) await new Promise(resolve => setTimeout(resolve, 10000)); // 4. Fulfill trade await xag.fulfillTrade(trade.hash, seller.did, "XRP", seller.seed); // 5. Log agent interaction await xag.log("Trade initiated", "info", buyer.did, buyer.seed); // 6. Wait for time-lock (if XRP escrow) await new Promise(resolve => setTimeout(resolve, 10000)); // 7. Fulfill trade await xag.fulfillTrade(trade.hash, seller.did, "XRP", seller.seed); await xag.log("Trade fulfilled successfully", "success", seller.did, seller.seed); // 8. Check reputation const reputation = await xag.getReputation(buyer.did); console.log(`Buyer reputation: ${reputation.score}`); // 9. Retrieve logs const logs = await xag.getLogs(buyer.did, 10); console.log(`Recent logs: ${logs.length}`); // 10. View transaction history const history = await xag.getTransactionHistory(buyer.did, 20); console.log(`Transaction history: ${history.length} transactions`); } finally { await xag.disconnect(); } }

XRPL Features Used

  • XLS-40 DIDs: Decentralized identifiers for agent identity
  • Native Escrows: Time-locked XRP escrows for trustless commerce
  • RLUSD Payments: Stablecoin payments via TrustLines
  • Transaction Memos: Immutable audit trail for all transactions
